# GENDY
Este es el proyecto final de la materia Ingenieria de Software I de la Universidad Nacional de Colombia. Gendy es una aplicacion realizada en PHP y HTML. El objetivo es conectar a una serie de negocios con sus respectivos clientes, en los que el cliente puede reservar una cita con un negocio. Guardando la fecha, hora, el servicio que desean y el valor del servicio. esta informacion es guardada en la base de datos.
